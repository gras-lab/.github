# gras-lab

> Building scalable, robust, and clean software — one commit at a time.

**gras-lab** is an engineering organization founded by [Alfio Spoto](https://github.com/nameisalfio) and [Gabriele Ruggieri](https://github.com/GabrieleRuggieri). We use this space as the technical foundation for our startup ventures and shared engineering portfolio.

---

## Mission

We build software that is maintainable, well-tested, and designed for scale. Our work is guided by a strict adherence to industry best practices — clean architecture, the DRY principle, meaningful code review, and continuous integration.
